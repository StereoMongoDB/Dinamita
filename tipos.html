MySQL

Es un sistema de gestión de bases de datos relacional de código abierto. Se utiliza principalmente para aplicaciones web y en proyectos que requieren alta disponibilidad y eficiencia en el manejo de datos estructurados. MySQL se trabaja con el lenguaje SQL (Structured Query Language) para realizar operaciones de inserción, consulta, actualización y eliminación de datos. Es ampliamente utilizado con tecnologías como PHP, Python, y JavaScript en el desarrollo de aplicaciones web.

PostgreSQL

Es una base de datos relacional de código abierto, conocida por su estabilidad, robustez y soporte de operaciones avanzadas como transacciones ACID y consultas complejas. PostgreSQL es utilizado en aplicaciones donde se requiere alta integridad de los datos, como sistemas bancarios, aplicaciones científicas, y proyectos empresariales grandes. Se trabaja también con SQL, pero ofrece soporte adicional para programación de funciones dentro de la base de datos.

MongoDB

Es una base de datos NoSQL orientada a documentos. En lugar de usar tablas y filas como en las bases de datos relacionales, almacena datos en documentos de formato JSON. MongoDB es ideal para trabajar con grandes volúmenes de datos no estructurados o semi-estructurados y es comúnmente utilizado en aplicaciones que requieren escalabilidad horizontal, como plataformas de redes sociales y aplicaciones móviles. Se maneja con el lenguaje de consulta MongoDB Query Language (MQL).

SQLite

Es una base de datos ligera y de código abierto que no requiere un servidor dedicado, lo que la hace perfecta para aplicaciones móviles y de escritorio. SQLite almacena datos en un archivo local, por lo que es fácil de integrar en proyectos pequeños o medianos, como aplicaciones móviles, sistemas embebidos y software de escritorio. Utiliza SQL como lenguaje de consulta, aunque es más limitada en características avanzadas que otros sistemas como MySQL o PostgreSQL.

Microsoft SQL Server

Es una base de datos relacional desarrollada por Microsoft. Está orientada a manejar grandes volúmenes de datos y se usa comúnmente en entornos empresariales. Es muy robusta y ofrece herramientas como SQL Server Management Studio para facilitar la administración y el desarrollo de bases de datos. Se utiliza principalmente en aplicaciones de empresas grandes que requieren una alta disponibilidad, seguridad, y escalabilidad. Se maneja utilizando T-SQL (Transact-SQL), una extensión de SQL creada por Microsoft.
